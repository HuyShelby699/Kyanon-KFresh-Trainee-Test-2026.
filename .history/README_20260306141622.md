1. Cài đặt XAMPP
2. Khởi động Apache
3. Tìm thư mục htdocs

Toàn bộ file web phải đặt trong thư mục:

C:\xampp\htdocs

Ví dụ:

C:\xampp\htdocs\Solution.php

Hoặc tạo folder project:

C:\xampp\htdocs\test\Solution.php
4. Tạo file PHP

Ví dụ tạo file:

index.php

Đường dẫn:

C:\xampp\htdocs\product-test\index.php

Dán code PHP của bạn vào.

Ví dụ test đơn giản trước:

<?php
echo "PHP is running with XAMPP!";
?>

Lưu lại.

5. Chạy trên trình duyệt

Mở trình duyệt và truy cập:

http://localhost/product-test/index.php

Nếu thấy dòng:

PHP is running with XAMPP!

→ PHP đã chạy thành công.
