RUN FILE WITH XAMPP WAY 
1. Install XAMPP

Download XAMPP from the official website:

https://www.apachefriends.org

Install XAMPP following the default setup.

2. Start Apache

Open XAMPP Control Panel and click:

Start → Apache

If Apache starts successfully, the status will turn green.

3. Locate the htdocs Folder

All web files must be placed inside the htdocs directory.

C:\xampp\htdocs

Example:

C:\xampp\htdocs\Solution.php

Or create a project folder:

C:\xampp\htdocs\test\Solution.php
4. Create a PHP File

Create a file, for example:

index.php

Example path:

C:\xampp\htdocs\product-test\index.php

Paste your PHP code into the file.

Example test code:

<?php
echo "PHP is running with XAMPP!";
?>

Save the file.

5. Run the Project in a Browser

Open your browser and go to:

http://localhost/product-test/index.php

If you see:

PHP is running with XAMPP!

then PHP is running successfully.
